# MycoDefensAI: AI Destekli Mantar Analizi

**Canlı Proje Linki:** https://iambertanates.pythonanywhere.com

**Geliştirici:** Bertan Ateş (Graphic Designer & AI Developer)

**Özet:** Makine öğrenmesi (Random Forest) algoritması kullanılarak mantarların fiziksel özelliklerine göre zehirli veya yenilebilir olduğunu tahmin eden web uygulaması. Özel veri setinin işlenmesinden modelin eğitilmesine, modern ve cam efektli (glassmorphism) arayüz tasarımından bulut sunucuda canlıya alınmasına (deployment) kadar tüm süreçler uçtan uca tamamlanmıştır.

**Kullanılan Teknolojiler:** Python, Scikit-learn, Pandas, NumPy, Flask, Joblib, HTML/CSS
